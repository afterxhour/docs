## Role Permissions

<CardGroup cols={3}>
  <Card title="Owner" icon="crown">
    - - Full workspace access
    - - Manage all settings
    - - Invite/remove collaborators
    - - All other permissions
  </Card>
  <Card title="Admin" icon="shield">
    - - Manage job positions
    - - Review all applications
    - - Configure webhooks
    - - View analytics
    - - All moderator permissions
  </Card>
  <Card title="Moderator" icon="user">
    - - Review applications
    - - Change application status
    - - View submission data
  </Card>
</CardGroup>

**Admin** is a role with many sensitive permissions. Ensure those who you add as an admin are trusted.
